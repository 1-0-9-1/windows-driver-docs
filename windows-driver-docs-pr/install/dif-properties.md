# DIF\_PROPERTIES


This DIF code is obsolete and no longer supported in Microsoft Windows 2000 and later versions of Windows.

To supply custom property pages for a device, an installer handles the [**DIF\_ADDPROPERTYPAGE\_ADVANCED**](dif-addpropertypage-advanced.md) request.
