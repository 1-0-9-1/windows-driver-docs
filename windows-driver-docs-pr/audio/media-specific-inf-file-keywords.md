# Media-Specific INF File Keywords


## <span id="media_specific_inf_file_keywords"></span><span id="MEDIA_SPECIFIC_INF_FILE_KEYWORDS"></span>


The list of INF file keywords at [INF File Sections and Directives](devinst.inf_file_sections_and_directives) contains keywords that are common to all device classes, but the following media-specific INF file keywords do not appear in the list:

[**SetupPreferredAudioDevices**](audio.setuppreferredaudiodevices)

The **SetupPreferredAudioDevices** keyword is supported in Windows 2000 and later. For more information, see [Media-Class INF Extensions](audio.media_class_inf_extensions).
