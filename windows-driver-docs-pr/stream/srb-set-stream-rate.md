# SRB\_SET\_STREAM\_RATE


## <span id="ddk_srb_set_stream_rate_ks"></span><span id="DDK_SRB_SET_STREAM_RATE_KS"></span>


The **SRB\_SET\_STREAM\_RATE** command sets the rate at which a stream should run.
