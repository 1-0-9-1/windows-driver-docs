Framework Driver Object

[This topic applies to UMDF 1.x.]

The framework driver object is exposed to drivers by the IWDFDriver interface. It is the framework representation of the driver image loaded in the driver host process. The framework creates a new driver object for each driver loaded in the driver host process. The IWDFDriver interface is passed to the driver by the IDriverEntry::OnInitialize method, which is the main entry point for the user-mode driver.
