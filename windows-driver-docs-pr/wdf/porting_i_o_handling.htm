Porting I/O

KMDF drivers handle I/O requests by creating one or more queues and associating one or more I/O event callback functions with each queue. To port a WDM driver’s I/O handling code to KMDF:

  • Port I/O queues.

  • Port I/O dispatch routines to I/O event callbacks.

  • Revise code that handles completed requests.

  • Revise Canceled Request Logic.

  • Revise Forward Request Logic.

  • Revise code that issues I/O requests.
