# Native 802.11 Miniport Driver Configuration


**Important**  The [Native 802.11 Wireless LAN](native-802-11-wireless-lan4.md) interface is deprecated in Windows 10 and later. Please use the WLAN Device Driver Interface (WDI) instead. For more information about WDI, see [WLAN Universal Windows driver model](wifi-universal-driver-model.md).

 

The following topics describe how the Native 802.11 miniport driver is configured to operate within a wireless LAN (WLAN):

[802.11 PHY Configuration](802-11-phy-configuration.md)

[Regulatory Domain Configuration](regulatory-domain-configuration.md)

[IHV Configuration Extensions](ihv-configuration-extensions.md)

[Configuration through a Network Profile](configuration-through-a-network-profile.md)
