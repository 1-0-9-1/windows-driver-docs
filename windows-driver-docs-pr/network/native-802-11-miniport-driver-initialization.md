# Native 802.11 Miniport Driver Initialization


**Important**  The [Native 802.11 Wireless LAN](native-802-11-wireless-lan4.md) interface is deprecated in Windows 10 and later. Please use the WLAN Device Driver Interface (WDI) instead. For more information about WDI, see [WLAN Universal Windows driver model](wifi-universal-driver-model.md).

 

The following topics describe the initialization requirements for a Native 802.11 miniport driver:

-   [Native 802.11 MiniportInitializeEx Requirements](miniportinitializeex-requirements.md)
-   [Native 802.11 Miniport Driver Registration](native-802-11-miniport-driver-registration.md)
