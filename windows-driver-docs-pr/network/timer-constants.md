# Timer Constants


## <a href="" id="ddk-timer-constants-ng"></a>


The **ControlTimeoutPeriod** for a USB Remote NDIS device is 10 seconds.

The **KeepAliveTimeoutPeriod** for a USB Remote NDIS device is 5 seconds.
