# Ntddndis.h


This header's topics includes topics for Windows networking device drivers starting with Windows Vista, as well as topics that are shared between network drivers and the user mode network applications that interact with them. 
The header file that contains the topics defined in this section are included in the Windows Software Development Kit (SDK).

> [!IMPORTANT]
> This header's topics include pages for definitions, macros, OIDs, and other data structures that are not part of standard network driver reference. 
For more information about standard network driver reference for these headers (structures, enumerations, functions, callbacks, and IOCTLs), see [Ntddndis.h (Reference)](https://msdn.microsoft.com/library/windows/hardware/mt808524).
