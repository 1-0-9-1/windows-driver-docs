# Native 802.11 Reset, Halt and Shutdown Operations


**Important**  The [Native 802.11 Wireless LAN](native-802-11-wireless-lan4.md) interface is deprecated in Windows 10 and later. Please use the WLAN Device Driver Interface (WDI) instead. For more information about WDI, see [WLAN Universal Windows driver model](wifi-universal-driver-model.md).

 

The following topics describe the operations performed by Native 802.11 miniport drivers when requested to reset, halt, or shutdown:

-   [MAC/PHY Reset](mac-phy-reset.md)
-   [Native 802.11 Miniport Driver Reset](driver-reset.md)
-   [Native 802.11 Miniport Driver Halt](driver-halt.md)
-   [Native 802.11 Miniport Driver Shutdown](driver-shutdown.md)
