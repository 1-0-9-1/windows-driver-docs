# MediaBackImageInfo element


The optional **MediaBackImageInfo** element contains the image information for the back side of an ADF duplex scan.

Usage
-----

``` syntax
<wscn:MediaBackImageInfo>
  child elements
</wscn:MediaBackImageInfo>
```

Attributes
----------

There are no attributes.
