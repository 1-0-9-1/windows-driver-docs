# ScannerElements element


The required **ScannerElements** contains all of the scanner information that is being returned to a client.

Usage
-----

``` syntax
<wscn:ScannerElements>
  child elements
</wscn:ScannerElements>
```

Attributes
----------

There are no attributes.
