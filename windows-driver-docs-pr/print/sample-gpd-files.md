# Sample GPD Files


## <a href="" id="ddk-sample-gpd-files-gg"></a>


The Windows Driver Kit (WDK) includes a large number of sample minidrivers. The samples include GPD files, along with font and resources files, for a variety of printer types. Samples are located in the \\src\\print\\mini subdirectory of the directory tree that contains the WDK.
