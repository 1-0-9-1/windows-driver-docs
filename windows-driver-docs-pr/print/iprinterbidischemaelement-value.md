# IPrinterBidiSchemaElement::Value method


The Value method returns the Bidi Schema element value.

Syntax
------

```ManagedCPlusPlus
HRESULT Value(
  [out, retval] VARIANT *pValue
);
```

Parameters
----------

*pValue* \[out, retval\]  
The returned element value.

Return value
------------

This method returns an **HRESULT** value.
