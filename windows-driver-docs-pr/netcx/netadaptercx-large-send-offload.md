# NetAdapterCx Large Send Offload

[!include[NetAdapterCx Beta Prerelease](../netcx-beta-prerelease.md)]

The TCP/IP transport supports large send offload (LSO). With LSO, the TCP/IP transport can offload the segmentation of large TCP packets for IPv4 and IPv6.
